# DateCalendarGenerator
This code takes a date from any year and returns which day of the week it was and the calendar month of that day fits into.<br/>
<br/>
ex.<br/>

Month: 12<br/>
Day: 24<br/>
Year: 2002<br/>
12 / 24 / 2002 is a <br/>
Tuesday  <br/>
